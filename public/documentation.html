Listado de bancos venezolanos

Has pruebas en tu aplicación con este listado de bancos existentes actualmente en Venezuela, obten información básica pero relevante de una entidad bancaria de manera sencilla y en un formato ligero(formato JSON).

API

Las peticiones a una URI devuelven un conjunto de objetos JSON con la siguiente estructura:
{
"id":integer,
"nombre":string,
"estado":boolean,
"codigo":string,
"privado":boolean,
"web":string,
"telefono":string,
"email":string
}

2.Tener solo un banco
Por medio de una petición HTTP GET accede a la URI https://venezuela-banks.herokuapp.com/api/banca/id, proporcionando el id numerico de un banco, si el id es correcto se devuelve un arreglo con un solo elemento en formato JSON con la siguiente estructura:
